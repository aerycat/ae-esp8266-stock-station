# ae-esp8266-stock-station
ESP8266 example base on [Arduino] IDE

功能简介：
  - 实时更新证券数据，该证券价格上涨LED显示为红色，反之为绿色。
  - 当前证券涨跌幅超2% LED慢速闪烁，超4% LED快速闪烁。
  - 串口监测打印相关系统信息。
